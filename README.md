# AgroMind 🌱

AgroMind is an AI-based smart agriculture system that helps farmers monitor crop health and soil conditions in real time.

## 🚀 Features
- 🌿 Leaf disease detection using image processing
- 📊 Soil analysis (pH, moisture, temperature)
- 📈 Fertility prediction based on soil data
- 💧 Smart irrigation support system

## 🧠 Tech Stack
- Python
- OpenCV
- Machine Learning (SVM)
- Streamlit
- ESP32 (for sensor data)

## ⚙️ How It Works
1. Capture or upload a leaf image
2. The model analyzes the leaf and detects diseases
3. Soil sensors send data (pH, moisture, etc.)
4. The system predicts soil fertility and suggests actions

## ▶️ How to Run
1. Clone the repository
2. Install required libraries:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
